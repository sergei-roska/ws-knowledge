# Dry Run: Performance Maintainability Drupal

## Example Task

Investigate slow listing pages after adding filtering and new theme interactions.

## Applied Workflow

1. Determine whether slowdown is cache, query/load, render, or asset related.
2. Gather evidence from query/load behavior and asset/render paths.
3. Prioritize optimizations by impact versus implementation risk.
4. Propose staged fixes with quick wins first.
5. Define pre/post metrics and rollback conditions.

## Expected Output

- Root-cause candidates with confidence level.
- Prioritized optimization plan with effort and impact.
- Deployment-safe sequence for implementation.
- Validation metrics for acceptance.
